Modelo de datos para plataforma comunitaria:

Entidad: Causa

Campos principales:
- id
- titulo
- descripcion
- fecha
- categoria

Propósito:
Almacenar las iniciativas comunitarias publicadas en la plataforma para que los
ciudadanos puedan conocerlas y decidir si desean brindarles su apoyo. Cada causa
tiene un identificador único que permite relacionarla con los apoyos que recibe.


Entidad: Apoyo

Campos principales:
- id
- causa_id
- usuario
- fecha

Propósito:
Registrar cada apoyo que recibe una causa comunitaria, vinculándolo con la causa
correspondiente a través del campo causa_id. Permite llevar un conteo de participación
y construir evidencia del respaldo ciudadano a cada iniciativa publicada.


Entidad: Usuario

Campos principales:
- id
- nombre_usuario
- correo
- contrasena_cifrada
- fecha_registro

Propósito:
Guardar la información de registro de los usuarios de la plataforma de forma segura,
permitiendo identificarlos al momento del login y controlar qué acciones pueden
realizar dentro del sistema según su perfil o rol asignado.
